# Student Resume Portfolio

This is a personal resume website built with HTML, CSS, and JavaScript. It includes sections for Home/About Me, Achievements, Projects, Skills, Resume (PDF download), and Contact.

## How to Use
- Replace placeholder text with your own information.
- Add your resume PDF to `assets/resume.pdf`.
- Customize styles in `style.css` as desired.

## Preview
Open `index.html` in your browser to view the site.
